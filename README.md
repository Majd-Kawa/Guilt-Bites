🍪 Guilt-Bites
📌 Project Overview

Guilt-Bites is a modern e-commerce web platform for browsing and ordering delicious baked treats online — including New York cookies, mini cookies, Ka’ak, and Maamoul.

It also features a Custom Occasion Box, allowing users to personalize orders for special events.

Users can explore products, add items to a cart, and place orders through an intuitive interface, while admins manage products and orders through a separate dashboard. The platform is built with scalability in mind for future enhancements.

✨ Key Features
🔐 User Authentication (Register & Login)
🛍️ Product Catalog with categories and details
🎁 Custom Occasion Box (personalized orders)
🛒 Shopping Cart & Order Management
🧑‍💼 Admin Dashboard (manage products, categories, orders)
📱 Responsive Design (mobile, tablet, desktop)
⚡ AJAX for dynamic updates
🗄️ MySQL Database for secure data storage

1. User Access

2. Product Browsing
Products are grouped into categories:
New York Cookies
Mini Cookies
Ka’ak
Maamoul
Custom Occasion Box
Each product includes images, descriptions, and price
3. Shopping
Users can view product details
Add items to the shopping cart
4. Checkout
Update or remove items from cart
Place order
Custom orders include:
Name
Event date
Design details
5. Admin Panel
Manage categories
Add/edit/delete products
